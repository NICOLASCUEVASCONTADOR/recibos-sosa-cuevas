Cómo conectar con Google Sheets

Seguí estos pasos una sola vez para activar la base de datos:

  1. Abrí drive.google.com y creá una nueva Google Spreadsheet
  2. En el menú, andá a Extensiones → Apps Script
  3. Borrá todo el contenido del editor y pegá el código del archivo Code.gs
  4. Guardá el proyecto (Ctrl+S) y ponerle un nombre, ej: "Recibos Sosa Cuevas"
  5. Hacé clic en Implementar → Nueva implementación
  6. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier usuario
  7. Hacé clic en Implementar, autorizá los permisos
  8. Copiá la URL de implementación y pegala acá abajo
